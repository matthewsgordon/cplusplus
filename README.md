# C++ Tutorial Program

This repository contains a simple C++ program in [helloworld.cpp](helloworld.cpp) that demonstrates how to define a class, create an object, assign values to its attributes, and print them to the console.

## What the program does

The program:
- defines a class named `MyClass`
- creates an object of that class
- sets an integer value and a string value
- prints both values to the terminal

## Compile the program

From the project directory, run:

```bash
g++ -std=c++11 helloworld.cpp -o helloworld
```

## Run the program

After compiling, run:

```bash
./helloworld
```

You should see output similar to:

```text
15
Some text
```
